# Ranger

[![Build Status](https://travis-ci.org/tdillon/ranger.svg?branch=master)](https://travis-ci.org/tdillon/ranger)

Range finder and plotter

```shell
$ ng build --prod --aot
$ gulp
```

Copy contents of ```dist/``` after build to root of web server.
